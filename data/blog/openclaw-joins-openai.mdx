### 一、发生了什么

除夕早上刷到一条爆炸新闻：[OpenClaw](https://steipete.me/posts/2026/openclaw) 的创始人 Peter Steinberger 宣布加入 OpenAI。

OpenClaw（前身是 Clawdbot / Moltbot）是一个开源的 AI Agent 框架，去年 11 月上线后迅速走红，GitHub star 数飙升，开发者社区反响强烈。和传统的聊天式 AI 不同，OpenClaw 是一个真正意义上的"自主代理"——它可以持续运行，代替用户清理收件箱、预订餐厅、办理航班值机，还能接入 WhatsApp、Slack 等消息平台。

Sam Altman 称 Steinberger 是"一个对高能力 Agent 如何协作完成任务有深刻见解的天才"。

### 二、为什么不自己开公司

这是很多人的第一反应：项目这么火，为什么不融资创业？

Steinberger 的回答很直接——他已经花了 13 年建立 PSPDFKit（一家 PDF SDK 公司），不想再重复一遍。他说："我想要的是改变世界，不是再建一家大公司。" 在他看来，OpenAI 是把 Agent 带给所有人的最快路径，因为这需要最前沿的模型、更成熟的安全体系，以及足够大的分发能力。

这个选择其实很务实。个人 Agent 不是一个纯技术问题，它涉及到安全、隐私、信任，这些东西单靠一个开源项目很难解决。Fortune 的报道里提到，已经有用户反馈 OpenClaw "失控"后自动发送了数百条 iMessage，网络安全专家也警告说，一个能访问私人数据、主动对外通信的 Agent 风险极高。

要做到"连我妈都能用"，光有技术不够，还需要体系。

### 三、OpenClaw 的命运

好消息是，OpenClaw 会保持开源。

项目将转移到基金会架构下运营，OpenAI 承诺赞助并允许 Steinberger 继续投入时间维护。Altman 也公开表态："未来一定是多 Agent 的。" OpenClaw 会继续支持多模型、多厂商，不会变成 OpenAI 的专属工具。

这个处理方式比较聪明。OpenAI 拿到了人才和社区影响力，OpenClaw 社区不会因为创始人的离开而崩盘，开发者也不用担心项目被收编。至少目前看，是一个双赢的局面。

### 四、行业信号

这件事背后有几个值得关注的信号：

**Agent 是下一个战场。** 从 OpenAI 到 Anthropic 到 Google，所有大厂都在押注 AI Agent。不再是"帮你写一段文字"，而是"帮你把事情办了"。Steinberger 的加入进一步印证了这个方向。

**开源 Agent 框架的窗口期可能在关闭。** 当顶级开源项目的创始人选择加入大厂而不是独立发展，这说明 Agent 领域的竞争已经进入了需要大量资源（模型、安全、分发）的阶段。纯社区驱动的项目会越来越难和大厂竞争。

**安全问题会成为 Agent 普及的最大瓶颈。** 一个能自主行动的 AI，如果没有足够的安全机制，后果可能比幻觉严重得多。OpenClaw 的"失控"事件就是一个预警。谁能先解决安全和信任问题，谁就能先把 Agent 推向主流。

### 五、写在最后

除夕看到这条新闻，感慨挺多的。

AI 行业的节奏太快了，一个项目从上线到走红到创始人被大厂收编，只用了三个月。这个速度在任何其他行业都是不可想象的。

对于我们这些也在做 AI 产品的人来说，这既是鼓舞也是警醒。方向是对的，Agent 一定是未来；但竞争也是残酷的，窗口期不会等人。

The claw is the law. 但 law 是谁定的，可能才是真正的问题。
